
## Movie API
API sederhana yang dibuat untuk mengelola data-data film

```
├── helper              # koneksi dan autentikasi
│   ├── connect.js
│   ├── auth.js
│   ├── verifyToken.js
├── documentation       # swagger dokumentasi
│   ├── documentation.json
├── routing             # perutean
│   ├── movie.js
│   ├── user.js
└── index.js
```


### Cara Menjalankan Program
```
npm run start
```
